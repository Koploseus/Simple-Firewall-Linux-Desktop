# Simple-Firewall-Linux-Desktop
Simple Linux Firewall for desktop, more feature in comming !

	-Flush IPTABLES
	-Allow DNS
	-Allow SSH OUTPUT
	-Allow loopback
	-Allow IRC Channel
	-Allow DHCP
	-Allow HTTP and HTTPS
	-Rejects Pings
	-Rejects Everything else
